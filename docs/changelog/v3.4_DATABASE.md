## [3.3.0] - 2024-01-20

### Added
- Database integration with comprehensive `request_logs` table
- Request logging for queued, completed, and failed requests
- Priority score tracking for all requests

### Changed
- Simplified database architecture using single `RequestRepository`
- Removed unnecessary repositories (ErrorRepository, MetricRepository, AnalyticsRepository)
- Updated imports to use only `get_request_repo` and `init_repositories`

### Technical Details
- All logging operations use `request_repo.log_request()` with different status values
- Comprehensive schema supports all required analytics:
  - Request rate analysis by IP/model
  - Performance metrics (queue wait, processing time)
  - Priority score distribution
  - Error rate and patterns
  - Model bunching detection

### Files Modified
- `src/smart_proxy.py` - Updated database logging calls
- `docs/DATABASE_INTEGRATION.md` - Updated documentation
