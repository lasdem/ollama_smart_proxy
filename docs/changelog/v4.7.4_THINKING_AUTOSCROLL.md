# v4.7.4 — Thinking panel autoscroll while streaming

**Date:** 2026-04-03

## Summary

The thinking `<pre>` uses `max-height` + `overflow-y: auto`. New tokens did not move the scroll position, so the visible area stayed at the top. After each update, `scrollTop` is set to `scrollHeight`. When the last chunk is `kind: content`, `full_thinking` is still applied to the thinking pane so it stays updated and scrolled.
